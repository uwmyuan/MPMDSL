# ORMDSL Code Generation Guide

This document introduces how to use ORMDSL to generate solver code in various formats.

---

## 1. Code Generation Overview

ORMDSL supports converting mathematical models to multiple solver formats:

| Target Format | Generator | Use Case |
|---------------|-----------|----------|
| AMPL | `AST2AMPL.scala` | Academic research |
| Gurobi | `IR2Gurobi.scala` | Industrial solving |
| Python | `ormdsl-py/` | Production environment |
| LaTeX | `AST2Tex.scala` | Paper display |

---

## 2. AMPL Code Generation

### 2.1 Using Scala Generator

```bash
cd ormdsl-intepreter/src
scalac AST2AMPL.scala
scala AST2AMPL
```

### 2.2 Generation Example

**Input AST:**

```scala
val formula = Formula(
  minimize = Minimize(sum(var_x[i], i in 1..n)),
  constraints = List(
    Constraint("c1", sum(a[i] * x[i]) <= b),
    Constraint("c2", x[i] >= 0)
  )
)
```

**Output AMPL:**

```ampl
# Model generated by ORMDSL

# Decision Variables
var x{1..n} >= 0;

# Objective
minimize obj: sum{i in 1..n} x[i];

# Constraints
subject to c1: sum{i in 1..n} a[i] * x[i] <= b;
subject to c2: x[i] >= 0;
```

### 2.3 AMPL Execution

```bash
ampl model.mod < model.run
```

## 3. Gurobi Code Generation

### 3.1 Using IR2Gurobi

```scala
import dsl.ormdsl_intepreter._

// Create IR
val formula = FormulaIR(
  declarations = List(
    IntegerDecisionVariable("x", ConstIR(0), ConstIR(100)),
    IntegerDecisionVariable("y", ConstIR(0), ConstIR(100))
  ),
  objective = MinimizeIR(AExpIR(VarIR("x"), PlusIR, VarIR("y"))),
  constraints = List(
    SimpleConstraint("c1", EquationIR(
      AExpIR(VarIR("x"), TimesIR, ConstIR(2)),
      LessEqIR,
      ConstIR(15)
    ))
  )
)

// Generate Gurobi code
val gurobiCode = IR2Gurobi.generateModel(formula)
```

### 3.2 Generated Code

```python
#!/usr/bin/env python3
# Generated by ORMDSL-IR2Gurobi

import gurobipy as gp
from gurobipy import GRB

def create_model():
    model = gp.Model('ormdsl_model')
    
    # Decision Variables
    x = model.addVar(lb=0, ub=100, vtype=GRB.INTEGER, name='x')
    y = model.addVar(lb=0, ub=100, vtype=GRB.INTEGER, name='y')
    
    # Objective
    model.setObjective(x + y, GRB.MINIMIZE)
    
    # Constraints
    model.addConstr(2*x <= 15, name='c1')
    
    model.optimize()
    return model

if __name__ == '__main__':
    create_model()
```

### 3.3 Direct Solving

```python
from ormdsl_gurobi import GurobiModel

model = GurobiModel("my_problem")
model.addVar("x", lb=0, ub=10, vtype="integer")
model.addConstraint("c1", "2*x <= 15")
model.setObjective("x", sense="minimize")

# Generate code
code = model.generatePythonCode()

# Or solve directly
result = model.solve()
```

## 4. LaTeX Formula Generation

### 4.1 Using AST2Tex

```scala
import dsl.ormdsl_intepreter._

val latex = AST2Tex.generate(problem.toAST)
println(latex)
```

### 4.2 Generation Example

**Input AST:**

```scala
Minimize(sum(c[i] * x[i], i in I))
Subject To
  sum(A[i,j] * x[i], i in I) <= b[j]  forall j in J
  x[i] >= 0  forall i in I
```

**Output LaTeX:**

```latex
\begin{align}
\min \quad & \sum_{i \in I} c_i x_i \\
\text{s.t.} \quad & \sum_{i \in I} A_{ij} x_i \leq b_j \quad \forall j \in J \\
                  & x_i \geq 0 \quad \forall i \in I
\end{align}
```

### 4.3 Complete Document Template

```latex
\documentclass{article}
\usepackage{amsmath}

\begin{document}

\section*{Optimization Model}

\subsection*{Parameters}
\begin{itemize}
\item $I$ - Set of customers
\item $J$ - Set of facilities
\item $c_i$ - Cost for customer $i$
\item $A_{ij}$ - Assignment coefficient
\item $b_j$ - Capacity of facility $j$
\end{itemize}

\subsection*{Decision Variables}
\begin{itemize}
\item $x_i \geq 0$ - Amount assigned from customer $i$
\end{itemize}

\subsection*{Mathematical Formulation}
\[
\begin{aligned}
\min \quad & \sum_{i \in I} c_i x_i \\
\text{s.t.} \quad & \sum_{i \in I} A_{ij} x_i \leq b_j \quad \forall j \in J \\
                  & x_i \geq 0 \quad \forall i \in I
\end{aligned}
\]

\end{document}
```

## 5. Python API

### 5.1 Installation

```bash
pip install ormdsl
```

### 5.2 Basic Usage

```python
from ormdsl import Model, Var, Constraint

# Create model
model = Model("knapsack")

# Add variables
x = model.addVars(["item1", "item2", "item3"], vtype="binary")

# Add constraints
model.addConstraint("capacity", 
    sum(weight[i] * x[i] for i in ["item1", "item2", "item3"]) <= 10)

# Set objective
model.setObjective(sum(profit[i] * x[i] for i in ["item1", "item2", "item3"]),
                   sense="maximize")

# Generate code
print(model.toGurobi())
```

### 5.3 Code Generation Options

```python
# Generate different formats
ampl_code = model.toAMPL()
gurobi_code = model.toGurobi()
latex_code = model.toLatex()

# Save to file
with open("model.py", "w") as f:
    f.write(gurobi_code)
```

## 6. Advanced Code Generation

### 6.1 Custom Variable Naming

```scala
val model = GurobiModel("custom_naming")
model.setVarNamePrefix("x_")
model.setConstrNamePrefix("c_")
```

### 6.2 Code Templates

```python
# Custom code template
template = """
# Custom Header
# Author: {author}
# Date: {date}

import gurobipy as gp
from gurobipy import GRB

def {model_name}():
    model = gp.Model('{model_name}')
    
    # Variables
    {variables}
    
    # Constraints  
    {constraints}
    
    # Solve
    model.optimize()
    return model

if __name__ == '__main__':
    {model_name}()
"""

code = model.generateFromTemplate(template)
```

### 6.3 Batch Generation

```python
# Batch generate from multiple problem instances
instances = ["small", "medium", "large"]

for inst in instances:
    model = load_instance(inst)
    code = model.toGurobi()
    
    with open(f"models/{inst}_model.py", "w") as f:
        f.write(code)
```

## 8. Code Generation Best Practices

### 8.1 Separate Data and Model

```python
# ❌ Avoid
model = Model()
model.addVar("x", lb=0, ub=100)  # Hard-coded

# ✅ Recommended
model = Model()
model.addVar("x", lb=lower_bound, ub=upper_bound)  # Parameterized
```

### 8.2 Use Configuration Files

```yaml
# config.yaml
model:
  name: "production_planning"
  type: "MIP"

solver:
  time_limit: 3600
  mip_gap: 0.001

output:
  format: "gurobi"
  path: "generated/"
```

```python
import yaml

with open("config.yaml") as f:
    config = yaml.safe_load(f)

model = Model(config["model"]["name"])
# ... apply config ...
code = model.toGurobi()
```

### 8.3 Version Control

```python
# Add metadata to generated code
header = f"""
# Generated by ORMDSL v{__version__}
# Timestamp: {datetime.now()}
# Target: Gurobi {gp.gurobi.version()}
"""
```

## 9. FAQ

### Q: Generated code has syntax errors?

A: Check:
- Variable names contain illegal characters
- Constraint expressions are complete
- Indentation is correct

### Q: How to handle nonlinear constraints?

A: Use piecewise linear approximation or quadratic constraints:

```python
# Nonlinear constraint
model.addQConstr(x * y <= 100)  # Quadratic constraint

# Or linearize
model.addConstr(z <= x)
model.addConstr(z <= y)
model.addConstr(z >= x + y - 1)
```

### Q: How to customize code format?

A: Override code generation templates:

```python
from ormdsl.generators import GurobiGenerator

class CustomGenerator(GurobiGenerator):
    def generate_header(self):
        return "# Custom Header\n"
    
    def format_variable(self, var):
        return f"# {var.name}: {var.vtype}"
```
