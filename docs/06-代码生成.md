# ORMDSL 代码生成指南

本文档介绍如何使用ORMDSL生成各种格式的求解器代码。

---

## 1. 代码生成概述

ORMDSL 支持将数学模型转换为多种求解器格式：

| 目标格式 | 生成器 | 用途 |
|----------|--------|------|
| AMPL | `AST2AMPL.scala` | 学术研究 |
| Gurobi | `IR2Gurobi.scala` | 工业求解 |
| Python | `ormdsl-py/` | 生产环境 |
| LaTeX | `AST2Tex.scala` | 论文展示 |

---

## 2. AMPL 代码生成

### 2.1 使用 Scala 生成器

```bash
cd ormdsl-intepreter/src
scalac AST2AMPL.scala
scala AST2AMPL
```

### 2.2 生成示例

**输入 AST:**

```scala
val formula = Formula(
  minimize = Minimize(sum(var_x[i], i in 1..n)),
  constraints = List(
    Constraint("c1", sum(a[i] * x[i]) <= b),
    Constraint("c2", x[i] >= 0)
  )
)
```

**输出 AMPL:**

```ampl
# Model generated by ORMDSL

# Decision Variables
var x{1..n} >= 0;

# Objective
minimize obj: sum{i in 1..n} x[i];

# Constraints
subject to c1: sum{i in 1..n} a[i] * x[i] <= b;
subject to c2: x[i] >= 0;
```

### 2.3 AMPL 运行

```bash
ampl model.mod < model.run
```

## 3. Gurobi 代码生成

### 3.1 使用 IR2Gurobi

```scala
import dsl.ormdsl_intepreter._

// 创建 IR
val formula = FormulaIR(
  declarations = List(
    IntegerDecisionVariable("x", ConstIR(0), ConstIR(100)),
    IntegerDecisionVariable("y", ConstIR(0), ConstIR(100))
  ),
  objective = MinimizeIR(AExpIR(VarIR("x"), PlusIR, VarIR("y"))),
  constraints = List(
    SimpleConstraint("c1", EquationIR(
      AExpIR(VarIR("x"), TimesIR, ConstIR(2)),
      LessEqIR,
      ConstIR(15)
    ))
  )
)

// 生成 Gurobi 代码
val gurobiCode = IR2Gurobi.generateModel(formula)
```

### 3.2 生成的代码

```python
#!/usr/bin/env python3
# Generated by ORMDSL-IR2Gurobi

import gurobipy as gp
from gurobipy import GRB

def create_model():
    model = gp.Model('ormdsl_model')
    
    # Decision Variables
    x = model.addVar(lb=0, ub=100, vtype=GRB.INTEGER, name='x')
    y = model.addVar(lb=0, ub=100, vtype=GRB.INTEGER, name='y')
    
    # Objective
    model.setObjective(x + y, GRB.MINIMIZE)
    
    # Constraints
    model.addConstr(2*x <= 15, name='c1')
    
    model.optimize()
    return model

if __name__ == '__main__':
    create_model()
```

### 3.3 直接求解

```python
from ormdsl_gurobi import GurobiModel

model = GurobiModel("my_problem")
model.addVar("x", lb=0, ub=10, vtype="integer")
model.addConstraint("c1", "2*x <= 15")
model.setObjective("x", sense="minimize")

# 生成代码
code = model.generatePythonCode()

# 或直接求解
result = model.solve()
```

## 4. LaTeX 公式生成

### 4.1 使用 AST2Tex

```scala
import dsl.ormdsl_intepreter._

val latex = AST2Tex.generate(problem.toAST)
println(latex)
```

### 4.2 生成示例

**输入 AST:**

```scala
Minimize(sum(c[i] * x[i], i in I))
Subject To
  sum(A[i,j] * x[i], i in I) <= b[j]  forall j in J
  x[i] >= 0  forall i in I
```

**输出 LaTeX:**

```latex
\begin{align}
\min \quad & \sum_{i \in I} c_i x_i \\
\text{s.t.} \quad & \sum_{i \in I} A_{ij} x_i \leq b_j \quad \forall j \in J \\
                  & x_i \geq 0 \quad \forall i \in I
\end{align}
```

### 4.3 完整文档模板

```latex
\documentclass{article}
\usepackage{amsmath}

\begin{document}

\section*{Optimization Model}

\subsection*{Parameters}
\begin{itemize}
\item $I$ - Set of customers
\item $J$ - Set of facilities
\item $c_i$ - Cost for customer $i$
\item $A_{ij}$ - Assignment coefficient
\item $b_j$ - Capacity of facility $j$
\end{itemize}

\subsection*{Decision Variables}
\begin{itemize}
\item $x_i \geq 0$ - Amount assigned from customer $i$
\end{itemize}

\subsection*{Mathematical Formulation}
\[
\begin{aligned}
\min \quad & \sum_{i \in I} c_i x_i \\
\text{s.t.} \quad & \sum_{i \in I} A_{ij} x_i \leq b_j \quad \forall j \in J \\
                  & x_i \geq 0 \quad \forall i \in I
\end{aligned}
\]

\end{document}
```

## 5. Python API

### 5.1 安装

```bash
pip install ormdsl
```

### 5.2 基本使用

```python
from ormdsl import Model, Var, Constraint

# 创建模型
model = Model("knapsack")

# 添加变量
x = model.addVars(["item1", "item2", "item3"], vtype="binary")

# 添加约束
model.addConstraint("capacity", 
    sum(weight[i] * x[i] for i in ["item1", "item2", "item3"]) <= 10)

# 设置目标
model.setObjective(sum(profit[i] * x[i] for i in ["item1", "item2", "item3"]),
                   sense="maximize")

# 生成代码
print(model.toGurobi())
```

### 5.3 代码生成选项

```python
# 生成不同格式
ampl_code = model.toAMPL()
gurobi_code = model.toGurobi()
latex_code = model.toLatex()

# 保存到文件
with open("model.py", "w") as f:
    f.write(gurobi_code)
```

```python

# Custom Header

# Date: {date}

import gurobipy as gp
from gurobipy import GRB

def {model_name}():
    model = gp.Model('{model_name}')
    
    # Variables
    {variables}
    
    # Constraints  
    {constraints}
    
    # Solve

    return model

    {model_name}()

```

### 6.3 批量生成


## 9. 常见问题

### Q: 生成的代码有语法错误？

A: 检查：
- 变量名是否包含非法字符
- 约束表达式是否完整
- 缩进是否正确

### Q: 如何处理非线性约束？

A: 使用分段线性近似或二次约束：

```python
# 非线性约束
model.addQConstr(x * y <= 100)  # 二次约束

# 或线性化
model.addConstr(z <= x)
model.addConstr(z <= y)
model.addConstr(z >= x + y - 1)
```

### Q: 如何自定义代码格式？

A: 重写代码生成模板：

```python
from ormdsl.generators import GurobiGenerator

class CustomGenerator(GurobiGenerator):
    def generate_header(self):
        return "# Custom Header\n"
    
    def format_variable(self, var):
        return f"# {var.name}: {var.vtype}"
```
