# 安装指南

本文档详细说明如何安装和配置 ORMDSL-Gurobi 模块及其依赖项。

---

## 📋 系统要求

| 组件 | 最低版本 | 推荐版本 |
|------|----------|----------|
| Python | 3.8 | 3.10+ |
| Gurobi | 9.0 | 13.0+ |
| 操作系统 | Windows 10 / Linux / macOS | Windows 11 / Ubuntu 22.04 |
| 内存 | 4 GB | 8 GB+ |
| CPU | 4 核心 | 8 核心+ |

---

## 🛠️ 安装步骤

### 步骤 1: 安装 Gurobi Optimizer

#### 方法 A: 通过 pip 安装 (推荐)

```bash
pip install gurobipy
```

#### 方法 B: 从官网下载安装

1. 访问 [Gurobi 下载页面](https://www.gurobi.com/downloads/)
2. 注册账号并申请学术许可证（如果是学生/教师）
3. 下载对应操作系统的安装包
4. 运行安装程序

#### 方法 C: Anaconda 环境

```bash
conda install -c gurobi gurobipy
```

### 步骤 2: 配置 Gurobi 许可证

#### 获取学术许可证（免费）

```bash
# 使用 grbgetkey 命令激活许可证
grbgetkey <your-license-key>
```

或在 [Gurobi 许可证中心](https://license.gurobi.com/) 在线获取。

#### 验证许可证

```python
import gurobipy as gp

# 检查许可证状态
print("许可证信息:")
print(gp.Environment().params)

# 验证许可证有效性
try:
    model = gp.Model()
    print("✓ Gurobi 许可证验证成功")
except Exception as e:
    print(f"✗ 许可证错误: {e}")
```

### 步骤 3: 安装 ORMDSL-Gurobi (可选)

如果需要使用 Scala 代码生成器:

```bash
# 安装 Scala
conda install -c conda-forge scala=2.13.12

# 或者使用 coursier
cs install scala:2.13.12
```

## 🔧 环境变量配置

### Windows

```powershell
# 设置 Gurobi 路径 (如果手动安装)
$env:GUROBI_HOME = "C:\gurobi1300\win64"
$env:PATH = "$env:GUROBI_HOME\bin;$env:PATH"

# 设置许可证路径 (可选)
$env:GRB_LICENSE_FILE = "C:\Users\<username>\gurobi.lic"
```

### Linux/macOS

```bash
# 在 ~/.bashrc 或 ~/.zshrc 中添加
export GUROBI_HOME=/opt/gurobi1300/linux64
export PATH=$GUROBI_HOME/bin:$PATH
export GRB_LICENSE_FILE=/home/<username>/gurobi.lic

# 使配置生效
source ~/.bashrc
```

## ✅ 验证安装

创建测试脚本 `verify_installation.py`:

```python
#!/usr/bin/env python3
"""验证 Gurobi 安装是否成功"""

import sys

def verify_installation():
    print("=" * 50)
    print("Gurobi 安装验证")
    print("=" * 50)
    
    # 1. 检查 Python 版本
    print(f"\n[1/5] Python 版本: {sys.version}")
    
    # 2. 尝试导入 gurobipy
    try:
        import gurobipy as gp
        print("[2/5] ✓ gurobipy 导入成功")
    except ImportError as e:
        print(f"[2/5] ✗ gurobipy 导入失败: {e}")
        return False
    
    # 3. 检查 Gurobi 版本
    version = gp.gurobi.version()
    print(f"[3/5] Gurobi 版本: {version[0]}.{version[1]}.{version[2]}")
    
    # 4. 检查许可证
    try:
        env = gp.Environment()
        print("[4/5] ✓ Gurobi 环境创建成功")
    except Exception as e:
        print(f"[4/5] ✗ 许可证或环境问题: {e}")
        return False
    
    # 5. 运行简单测试
    try:
        model = gp.Model("test")
        x = model.addVar(vtype=GRB.CONTINUOUS, name="x")
        model.setObjective(x, GRB.MAXIMIZE)
        model.addConstr(x <= 10)
        model.optimize()
        if model.status == GRB.OPTIMAL:
            print(f"[5/5] ✓ 简单模型求解成功 (x = {x.x})")
        model.dispose()
    except Exception as e:
        print(f"[5/5] ✗ 模型求解失败: {e}")
        return False
    
    print("\n" + "=" * 50)
    print("✓ 所有验证通过！Gurobi 安装成功！")
    print("=" * 50)
    return True

if __name__ == "__main__":
    from gurobipy import GRB
    success = verify_installation()
    sys.exit(0 if success else 1)
```

运行验证:

```bash
python verify_installation.py
```

## ❓ 常见问题

### Q: 许可证过期怎么办？

A: 访问 [Gurobi 许可证中心](https://license.gurobi.com/) 续期或申请新许可证。

### Q: 如何在不同机器间迁移许可证？

A: 学术许可证支持最多 3 台机器。运行 `grbgetkey` 获取新机器的许可证。

### Q: 许可证文件路径？

A: 默认位置:
- Windows: `C:\Users\<username>\gurobi.lic`
- Linux: `/home/<username>/gurobi.lic`
- macOS: `/Users/<username>/gurobi.lic`
